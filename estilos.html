Modificar Estilos CSS

En esta página aprenderemos a modificar los estilos CSS de una página web.

Podemos cambiar colores, fuentes, tamaños, márgenes, entre otros aspectos visuales.

Los estilos CSS se aplican a través de selectores que apuntan a los elementos HTML.

Código CSS de ejemplo

.ejemplo {
    color: blue;
    font-size: 20px;
    text-align: center;
}
.ejemplo_p{
    color: red;
    font-size: 16px;
}
